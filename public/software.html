可视化

Package downloads github publication
circlize：圈形可视化

HilbertCurve：希尔伯特曲线布局
spiralize：阿基米德螺线布局
gtrellis：基因组Trellis布局
BioCartaImage：BioCarta生物通路图像

数据建模与分析

Package downloads github publication
sfcurve：nxn 空间填充曲线
simona：生物本体论（Bio-ontology）语义相似性工具箱
simplifyEnrichment：简化功能富集分析结果
cola：一致性聚类通用框架
rGREAT：基于基因组区间的功能富集
CePa：基于网络中心性的生物通路富集
